# fubs-os

My own Universal Blue image, based on [aurora](https://github.com/ublue-os/aurora), with several modifications:
- most packages from Aurora-DX but
  - no Visual Studio Code
  - no Docker runtime
- Pre-release version of OpenConnect (with fix to connection issues with our VPN).
